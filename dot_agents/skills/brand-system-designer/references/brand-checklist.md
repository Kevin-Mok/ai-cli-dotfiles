# Brand System Checklist

## Discovery
- [ ] Product name and tagline confirmed
- [ ] Audience and use case documented
- [ ] Tone, personality, and values defined
- [ ] Constraints and platform targets identified
- [ ] Existing brand references reviewed

## Concept Direction
- [ ] Three distinct directions drafted
- [ ] Risks/tradeoffs captured per direction
- [ ] One direction selected with rationale

## Visual Outputs
- [ ] SVG wordmark/lockup produced
- [ ] SVG icon mark produced
- [ ] Favicon SVG produced
- [ ] Monochrome variation considered
- [ ] Logo reads clearly at small size

## Brand System
- [ ] Palette token set written (primary, secondary, accent, background, foreground)
- [ ] Typography pairing proposed with rationale
- [ ] Usage notes for spacing and minimum size added
- [ ] do/don't guidance included
- [ ] OG/social usage guidance noted (if relevant)

## Validation
- [ ] Repo path conventions checked
- [ ] File naming matches style used in repo
- [ ] Contrast and contrast-related warnings documented
- [ ] Open questions and follow-up options listed
