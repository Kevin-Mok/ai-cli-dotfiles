# Logo Evaluation Rubric

Rate each proposal (1-5) with short notes.

## Distinctiveness
- Is the concept memorable and not generic?

## Clarity at small sizes
- Is the primary shape legible at 16px and 32px?

## Scalability
- Does it survive scaling from favicon to hero use?

## Monochrome performance
- Is it still understandable in single-color use?

## Technical fit
- Is the mark aligned with technical/product context?
- Is it safe for code and documentation usage?

## Color system quality
- Is the palette balanced across UI contexts?
- Can it support light and dark presentation?

## Implementation readiness
- Are files usable as SVG and easy to integrate?
- Are tokens and usage rules explicit enough for handoff?

## Final score
Score each dimension and calculate a total out of 35.
- 30-35: Strong
- 24-29: Good
- 18-23: Needs refinements
- <18: Rework required
